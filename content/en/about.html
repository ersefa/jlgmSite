About this site

Besides a resume and contact showcase this site is like a sandbox where I can apply new tricks learnt in web development.

For those interested in "under the hood" stuff (back-end):

This site was built with nanoc, an awesome Ruby framework created by Denis Defreyne.
nanoc is a tool that runs on your local computer and compiles documents written in formats such as Markdown, Textile, Haml… into a static web site consisting of simple HTML files.

For those interested in user experience and design (front-end):

This site uses HTML5 semantics also some CSS3 new properties like transformations, @face fonts, media queries, shadows, etc.

This site was built applying responsive and adaptative web design thanks to the new media query CSS3 property. This site can adaptate to any smartphone, tablet or mobile device using only one cascade style sheet without specific development for those devices. In fact you can play with it resizing your navigation window.
