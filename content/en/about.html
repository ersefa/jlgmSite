About this site

This site is built with nanoc, an awesome Ruby framework created by Denis Defreyne.
nanoc is a tool that runs on your local computer and compiles documents written in formats such as Markdown, Kramdown, Textile, Haml… into a static web site consisting of simple HTML files.

Feel free to check the code at GitHub
