Acerca de esta pagina

Esta página además de una forma para darme a conocer, mostrar mi curriculum y aportar una forma de contacto, pretende ser un lugar donde pueda poner en práctica los conocimientos que voy adquiriendo en desarrollo web, un sitio donde "jugar" con nuevas tecnologías como pueden ser CSS3, HTML5, etc.

Para aquellos que les interesa lo que hay "under the hood" (back-end):

La página ha sido creada con nanoc, un potente framework para Ruby creado por Denis Defreyne.
Gracias a nanoc podemos compilar desde nuestra maquina local documentos escritos en diversos formato, como por ejemplo Markdown, Kramdown, Textile, Haml y convertirlo en una página estática formada sólo por HTML.

Utiliza nanoc nos permite utilizar Ruby, en mi caso concreto lo utilizo para automatizar ciertas tareas como por ejemplo la de facilitar la presentación en diferentes idiomas

Para aquellos mas interesados en la experiencia de usuario, en la parte estética (front-end):

En esta página se utiliza semántica HTML5 acompañada de diferentes propiedades CSS introducidas en su versión 3. Por citar unas cuantas: transformaciones, web fonts, media queries, sombras, etc.

Mención especial para las media queries, que me han permitido crear una página que pueda adaptarse y responder a los diferentes dispositivos donde puede ser vista, ya sean smartphones, tablets, ordenadores, etc. Esta forma de desarrollo se puede denominar como "Responsive and Adaptative web Design", y se basa en la filosofía de no realizar páginas especiales para dispositivos móviles sino que gracias a CSS3 podemos adaptar la misma en tiempo real para cada dispositivo.
