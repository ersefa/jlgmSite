Acerca de esta pagina

Esta página además de una forma para darme a conocer, mostrar mi curriculum y aportar una forma de contacto, pretende ser un lugar donde pueda poner en práctica los conocimientos que voy adquiriendo en desarrollo web, un sitio donde "jugar" con nuevas tecnologías como pueden ser CSS3, HTML5, etc.

Para aquellos que les interesa lo que hay "under the hood" (back-end):

La página ha sido creada con nanoc, un potente framework para Ruby creado por Denis Defreyne.
Gracias a nanoc podemos compilar desde nuestra maquina local documentos redactados en diversos formato, como por ejemplo Markdown, Textile, Haml y convertirlo en una página estática formada sólo por HTML.

nanoc nos permite utilizar Ruby, en mi caso concreto lo utilizo para automatizar ciertas tareas como por ejemplo la de facilitar la presentación en diferentes idiomas.

Para aquellos mas interesados en la experiencia de usuario, en la parte estética (front-end):

En esta página se utiliza semántica HTML5 acompañada de diferentes propiedades CSS introducidas en su versión 3. Por citar unas cuantas: transformaciones, web fonts, media queries, sombras, etc.

Mención especial merecen las media queries las cuales me han permitido crear una página que puede adaptarse y responder a los diferentes dispositivos donde puede ser vista ya sean smartphones, tablets, ordenadores, etc. Esta forma de desarrollo se denomina "Responsive and Adaptative web Design", y consiste en adaptar tanto los contenidos como el diseño de la página a cualquier tipo de tamaño o resolución de pantalla. Estos cambios se realizan gracias a las mencionadas media queries y permiten que la página se adapte en tiempo real y sin necesidad de peticiones complementarias al servidor. De esta forma se hace innecesario crear una página web adicional dedicada únicamente a dispositivos móviles. De hecho, si te apetece, puedes probar a redimensionar tu navegador en cualquier sección de la página para que veas dicha adaptación.
